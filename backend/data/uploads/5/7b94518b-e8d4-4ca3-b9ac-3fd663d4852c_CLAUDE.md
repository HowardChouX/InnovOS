When working in those subdirectories, first check if a `CLAUDE.md` exists in that project root and follow its guidance.

### Claude Code Provider Switcher

Switch between AI providers for Claude Code:
```bash
switch-claude          # Interactive selection
switch-ch              # CherryIn provider
switch-sf              # SiliconFlow provider
switch-ef              # EdgeFn provider
```
The script (located in `~/.local/bin/`) writes provider config to `~/.config/claude-provider`.

## Hyprland (Window Manager)

Config: `~/.config/hypr/` (see `README.md` there for full docs)

Essential shortcuts:
| Key | Action |
|-----|--------|
| `Super+Q` | Terminal (Kitty) |
| `Super+R` | Launcher (Wofi) |
| `Super+E` | File manager (Dolphin) |
| `Super+C` | Close window |
| `Super+F` | Fullscreen |
| `Super+1-0` | Workspace 1-10 |
| `Print` | Screenshot |

Scrolling layout (v0.54+ default):
| Key | Action |
|-----|--------|
| `Super+Alt+H/L` | Move window left/right column |
| `Super+Alt+J/K` | Scroll down/up |

Media keys: `F11/F12` brightness, `Super+=/-` volume, `Super+B` play/pause.

### Common Hyprland issues

- **Clipboard not working**: `systemctl --user restart wl-clipboard` or restart `hyprland-clipboard`
- **Screen sharing fails**: Check `systemctl --user status xdg-desktop-portal-hyprland`; restart if needed.
