# 管理员后台

> 对应思维导图节点：管理员附加（专利数据库管理、数据监控、模型服务、用户管理）

## 1. 概述

管理员后台提供系统管理功能，仅 `role='admin'` 的用户可访问。

```
管理员附加
├── 专利数据库管理 → 导入/编辑/删除专利数据
├── 数据监控 → 参考 CherryStudio 设置页面的 Web 服务接口管理
├── 模型服务 → AI 供应商 + Key 融合管理（参考 CherryStudio 设置页面）
└── 用户管理 → 用户列表、角色分配
```

## 2. 专利数据库管理

### 2.1 功能

- 导入专利数据（JSON/CSV 批量导入）
- 编辑专利信息
- 删除专利
- 专利向量化索引管理

### 2.2 API 端点

```
GET    /api/admin/patents             # 专利列表
POST   /api/admin/patents/import      # 批量导入
PUT    /api/admin/patents/{id}        # 编辑
DELETE /api/admin/patents/{id}        # 删除
POST   /api/admin/patents/reindex     # 重新索引向量
```

### 2.3 前端组件

```
frontend/src/features/admin/PatentDbPage.tsx
├── 专利列表表格
├── 导入对话框（支持 JSON/CSV）
└── 索引状态指示
```

## 3. 数据监控

### 3.1 功能

思维导图标注「参考 CherryStudio 的设置页面进行 Web 服务接口的管理」。

借鉴 CherryStudio 的 `SettingsWindowService` 模式：
- 系统健康检查仪表板
- API Key 使用统计
- 任务执行统计
- 数据库存储统计

### 3.2 监控面板

```
GET /api/monitor/health     # 系统健康检查
GET /api/monitor/stats      # 统计数据
GET /api/monitor/key-usage  # Key 使用情况
```

监控面板内容：
- **系统状态**：数据库延迟、内存使用、磁盘空间
- **AI API 状态**：各 Key 的调用次数、成功率、平均延迟
- **任务统计**：今日任务数、完成率、平均耗时
- **用户统计**：活跃用户数、注册趋势

### 3.3 前端组件

```
frontend/src/features/admin/MonitorPage.tsx（现有，重构）
├── OverviewCards.tsx      # 概览卡片
├── KeyUsageChart.tsx      # Key 使用图表
├── TaskStatsChart.tsx     # 任务统计图表
├── SystemStatus.tsx       # 系统状态
└── HealthCheckPanel.tsx   # 健康检查
```

## 6. 路由结构

```python
# backend/app/api/admin/__init__.py
from fastapi import APIRouter
router = APIRouter(prefix="/api/admin", tags=["admin"])

from . import keys, users, monitor, patent_db
router.include_router(keys.router)
router.include_router(users.router)
router.include_router(monitor.router)
router.include_router(patent_db.router)
```
