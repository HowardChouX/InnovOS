### 1.2 架构设计

思维导图标注「专利的数据信息和知识库一样都是类似的（CherryStudio 部分）」，意味着专利检索应复用知识库的 RAG 检索架构：

```python
# 专利检索复用知识库检索管线
class PatentSearchEngine:
    """专利检索引擎 — 复用知识库 RAG 架构。"""

    def __init__(self):
        self.chunker = TextChunker(chunk_size=256)  # 专利摘要较短
        self.embedder = Embedder()
        self.vector_store = VectorStore()

    async def index_patents(self, patents: list[dict]):
        """索引专利数据。"""
        for patent in patents:
            text = f"{patent['title']} {patent['abstract']} {patent['claims']}"
            chunks = self.chunker.chunk(text)
            vectors = await self.embedder.embed([c["text"] for c in chunks])
            metadata = [{"patent_id": patent["id"], **c} for c in chunks]
            self.vector_store.add(vectors, metadata)

    async def search(self, query: str, top_k: int = 10) -> list[dict]:
        """语义检索相关专利。"""
        query_vector = (await self.embedder.embed([query]))[0]
        return self.vector_store.search(query_vector, top_k)
```

## 2. 专利转化

### 2.1 功能概述

```
专利转化
├── 检索专利数据库对用户输入的方案进行专利转化
```

```sql
CREATE TABLE patents (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    patent_number TEXT UNIQUE,
    title TEXT NOT NULL,
    abstract TEXT,
    claims TEXT,
    assignee TEXT,
    filing_date TEXT,
    category TEXT,
```
