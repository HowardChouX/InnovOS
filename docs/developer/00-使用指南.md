# 使用指南

## 系统简介

InnovOS（智融创新操作系统）是一个 AI 驱动的创新问题求解平台，通过多 Agent 工作流帮助用户完成从需求洞察到方案评估的全流程。

## 快速开始

### 环境要求

- Python 3.10+
- Node.js 18+（推荐 20）
- npm 或 pnpm

### 启动开发环境

```bash
# 1. 安装后端依赖
cd backend && uv sync

# 2. 安装前端依赖
cd frontend && npm install

# 3. 同时启动前后端
make dev

# 或分别启动：
make backend   # FastAPI → http://localhost:8000
make frontend  # Vite   → http://localhost:5173
```

### 首次使用

1. 访问 `http://localhost:5173/register` 注册账户
2. 使用注册的账户登录
3. 在「模型服务」页面添加至少一个 AI API Key（如 DeepSeek）
4. 点击「首页」→「新建任务」开始创新分析

## 主要功能模块

### 普通用户模块

| 模块 | 说明 |
|------|------|
| 首页 | 任务概览、快捷入口、系统状态 |
| 知识库 | 管理个人知识文档，支持 RAG 增强检索 |
| 流程分析 | 6 阶段 AI 工作流：需求画像 → 问题建模 → 专利检索 → 方案生成 → 方案评估 |
| 历史方案 | 查看和管理历次分析的完整数据 |
| 专利转化 | 检索专利数据库，生成专利转化方案 |

### 管理员模块

| 模块 | 说明 |
|------|------|
| 模型服务 | 添加/编辑/删除 AI API Key，设置轮询优先级和限流 |
| 用户管理 | 查看用户列表、角色分配 |
| 数据监控 | 系统健康检查、Key 使用统计、任务统计 |
| 专利数据库管理 | 导入/管理专利数据源 |

## 侧边栏状态机

工作流运行时，侧边栏实时显示当前 Agent 管线进度：

```
[需求画像] → [问题建模] → [专利检索] → [方案生成] → [方案评估]
    ↓            ↓            ↓            ↓            ↓
  进行中       等待         等待         等待         等待
```

## 账户与权限

- **普通用户**：可访问公共模块（首页、知识库、流程分析、历史方案、专利转化）
- **管理员**：额外可访问模型服务、用户管理、数据监控、专利数据库管理
- JWT Token 有效期 24 小时，通过 `Authorization: Bearer <token>` 头传递

## API 端点概览

基础路径：`http://localhost:8000/api`

| 端点 | 方法 | 说明 |
|------|------|------|
| `/auth/register` | POST | 用户注册 |
| `/auth/login` | POST | 用户登录 |
| `/tasks` | GET/POST | 任务列表/创建 |
| `/knowledge/docs` | GET/POST | 知识库文档 |
| `/knowledge/search` | GET | 知识检索（RAG） |
| `/notifications` | GET | 通知列表 |
| `/notifications/unread-count` | GET | 未读通知数 |
| `/workflow/{task_id}/run` | POST | 启动工作流 |
| `/workflow/{task_id}/status` | GET | 工作流状态 |
| `/patents/search` | GET | 专利检索 |
| `/admin/keys` | GET/POST | 模型服务管理 |
| `/admin/users` | GET | 用户管理 |
| `/monitor/health` | GET | 系统健康检查 |
