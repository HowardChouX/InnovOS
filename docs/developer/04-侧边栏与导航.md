# 侧边栏与导航

> 对应思维导图节点：侧边栏（普通用户公共模块、管理员附加）

## 1. 三栏布局概览

InnovOS 采用三栏布局：

```
┌─────────┬───────────────────────────┬──────────────────┐
│ Sidebar │      Main Content         │ 流程状态机面板    │
│ (左栏)   │      (中栏)               │ (右栏)           │
│         │                           │                  │
│ 首页    │   ← 当前步骤结果展示       │ ① 需求洞察       │
│ 知识库  │     选中任务后显示          │ ② 问题建模       │
│ 历史方案 │     分析结果/详情          │ ③ 专利检索       │
│ 专利检索 │     编辑界面等             │ ④ 方案生成       │
│ 专利转化 │                           │ ⑤ 方案评估       │
│         │                           │   状态: 等待中    │
│ [管理员] │                           │                  │
└─────────┴───────────────────────────┴──────────────────┘
```

**布局规则：**
- **左栏 Sidebar** — 固定导航，始终显示
- **中栏 Main Content** — 路由页面内容（`<Outlet />`），根据当前路由显示对应模块
- **右栏 流程状态机面板** — 仅工作流运行时显示于首页，展示 5 步骤进度；用户评分交互也在此面板中

## 2. 侧边栏结构

```
侧边栏
├── 普通用户（公共模块）
│   ├── 首页
│   ├── 知识库
│   ├── 历史方案
│   ├── 专利检索
│   └── 专利转化
└── 管理员附加（仅 admin 显示）
    ├── 模型服务
    ├── 用户管理
    ├── 数据监控
    └── 专利数据库管理
```

> **注意：工作流步骤不放在侧边栏中。** 5 个流程步骤（需求洞察→问题建模→专利检索→方案生成→方案评估）显示在右侧的流程状态机面板中。

## 3. 普通用户公共模块

| 路径 | 组件 | 说明 |
|------|------|------|
| `/` | `DashboardPage` | 首页仪表板 |
| `/knowledge` | `KnowledgeBasePage` | 知识库管理 |
| `/history` | `HistoryPage` | 历史方案浏览 |
| `/patents` | `PatentSearchPage` | 专利检索 |
| `/patent-conversion` | `PatentConversionPage` | 专利转化 |

## 4. 管理员附加模块

| 路径 | 组件 | 说明 |
|------|------|------|
| `/admin/keys` | `KeyManagementPage` | 模型服务（AI 供应商 + Key 池管理） |
| `/admin/users` | `UserManagementPage` | 用户管理（列表/角色/删除） |
| `/monitor` | `MonitorPage` | 数据监控面板（系统/Key/任务统计） |
| `/admin/patents` | `PatentDbPage` | 专利数据库管理（规划中） |

管理员模块仅在 `role === 'admin'` 时显示在侧边栏底部「管理员」区域。

## 5. 导航配置

见 `frontend/src/utils/constants.ts`：

```typescript
export const ROUTES = {
  HOME: '/',
  PATENTS: '/patents',
  KNOWLEDGE: '/knowledge',
  MONITOR: '/monitor',
  HISTORY: '/history',
  PATENT_CONVERSION: '/patent-conversion',
  ADMIN_KEYS: '/admin/keys',
  ADMIN_USERS: '/admin/users',
} as const;

export const NAV_ITEMS = [
  { label: '首页', path: ROUTES.HOME, icon: 'fa-house' },
  { label: '知识库', path: ROUTES.KNOWLEDGE, icon: 'fa-book' },
  { label: '历史方案', path: ROUTES.HISTORY, icon: 'fa-clock-rotate-left' },
  { label: '专利检索', path: ROUTES.PATENTS, icon: 'fa-file-alt' },
  { label: '专利转化', path: ROUTES.PATENT_CONVERSION, icon: 'fa-file-contract' },
];
```
