# InnovOS 智能创新问题求解系统

## 系统介绍

InnovOS 是一款基于全球专利数据库、真实创新案例数据库及人工智能算法构建的智能化问题求解系统。系统面向技术研发、工程优化、成果转化及创新设计等应用场景，通过对海量专利、技术文献及创新案例进行结构化清洗、语义建模与路径抽取，构建创新问题知识数据库，并结合自主研发的 ZR-IPM（智融创新问题映射）算法，实现从用户需求输入到成果转化辅助的全流程智能化支持。

## 核心价值

### 核心算法
- **ZR-IPM 算法** - 创新问题智能建模，准确率 87.4%
- **多 Agent 协同** - ZR-MoA 多模型动态路由 + 专业分工
- **知识增强** - Patent-RAG，专利路径有效召回率 86.1%
- **质量评估** - 四维评估引擎，无效方案生成率 9.1%

### 技术特点
- **数据支撑** - 10 万+ 专利、3000+ 场景案例、5000+ 企业问题
- **智能语义检索** - 向量语义检索 + 相似路径匹配
- **闭环学习** - 用户反馈学习 + 人机交互测试 + 案例回流优化
- **高性能** - 平均响应时间 23.6 秒，Token 消耗降低 32.8%

## 项目目标

1. 构建完整的创新问题知识库和路径推理系统
2. 实现问题识别准确率 ≥88%
3. 实现相似路径匹配准确率 ≥85%
4. 实现方案生成成功率 ≥90%
5. 支持多用户协作和企业管理
6. 建立完整的数据导出和权限管理系统

## 技术栈

### 后端
- **Web 框架** - FastAPI (Python 3.10+)
- **数据库** - SQLite（开发）/ PostgreSQL（生产）
- **认证** - JWT (python-jose) + bcrypt
- **AI 模型** - DeepSeek-R1, Qwen-Turbo, Qwen-Max, BGE-M3

### 前端
- **框架** - React 19 + TypeScript
- **构建工具** - Vite 8
- **UI 框架** - TailwindCSS 4 + 自定义组件库
- **状态管理** - Zustand 5
- **路由** - React Router v7

### AI 架构（ZR-MoA）
- **主模型** - DeepSeek-R1（复杂推理）
- **专业模型** - Qwen-Turbo（需求洞察）、BGE-M3（向量化）、Qwen-Max（生成）
- **增强机制** - Patent-RAG 向量检索增强
