## 2. 核心架构机制

### 2.1 ZR-MoA 多模型协同架构

```
                    ┌──────────────────┐
                    │   主模型         │
                    │  DeepSeek-R1    │
                    │  (长链推理)      │
                    └────────┬─────────┘
                             │ 任务分发
         ┌───────────────────┼───────────────────┐
         ▼                   ▼                   ▼
┌─────────────────┐ ┌─────────────────┐ ┌─────────────────┐
│  需求洞察       │ │  专利分析       │ │  方案生成       │
│  Qwen-Turbo    │ │  BGE-M3        │ │  Qwen-Max      │
│  (NLP理解)      │ │  (向量化+检索)  │ │  (长文本生成)   │
└─────────────────┘ └─────────────────┘ └─────────────────┘
         │                   │                   │
         └───────────────────┼───────────────────┘
                             ▼
                    ┌──────────────────┐
                    │   Embedding +    │
                    │   RAG 知识库     │
                    │   10万+ 专利     │
                    └──────────────────┘
```

### 2.2 多 Agent 工作流

```
用户输入 → 需求洞察Agent → 问题建模Agent → 专利分析Agent
              → 方案生成Agent → 方案评估Agent → 成果转化Agent
```

每个 Agent 独立运行，通过工作流引擎协同，支持异步执行和状态追踪。

## 3. 数据流

### 3.1 前端数据流

```
组件 → Zustand Store → api/*.ts → fetch → FastAPI → SQLite
```

### 3.2 后端请求生命周期

```
HTTP Request → CORS中间件 → Router → Depends(get_current_user)
  → get_db() → SQL Query → Response
```

## 4. 部署架构

| 环境 | 数据库 | 前端 | 后端 |
|------|--------|------|------|
| 开发 | SQLite | Vite Dev Server (:5173) | Uvicorn (:8000) |
| 测试 | SQLite 测试库 | Vite Build + Serve | Uvicorn |
| 生产 | PostgreSQL | Nginx + Static | Gunicorn + Uvicorn |

## 5. 技术选型

| 层次 | 技术 | 说明 |
|------|------|------|
| 前端框架 | React 19 + TypeScript | 组件化 UI |
| 构建工具 | Vite 8 | 快速 HMR |
| 样式 | TailwindCSS 4 | 原子化 CSS |
| 状态管理 | Zustand 5 | 轻量 Store |
| 后端框架 | FastAPI | 高性能异步 |
| 数据库 | SQLite | 结构化存储 |
| 认证 | JWT (python-jose + bcrypt) | 24h 过期 |
